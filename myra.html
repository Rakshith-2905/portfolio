Project MYRA

An Educational Robotics platform

About the Project

The objective of this project was to develop an open-source, low-cost robot that is designed to be built and programmed for various functions by students ranging from middle school through PhD scholars.

The robot is 80% 3D printed and 20% laser cut. The aim of the project is to opensource the files of the project and create a sustaning curriculum which educated students on robotics at various depths

Skills Involved

  • 3D Printing
  • OpenCV Image Processing
  • Face and Emotion Recognition
  • Python
  • Embedded controller programming in c++
  • WireLess Interface
  • USB Serial Protocol
  • Forward and Inverse Kinematics
  • Front End development - HTML
  • PCB Designing

About

The Robot is demonstrating its ability to track human face and move its eyes accordingly to hold the gaze of the human.
The robot has three level of controllers. The master controller is a Jetson TX2 it is used for performing high end image processing tasks and host the touch interface. An ATMEL chip is used as a low level controller which mearly operates as a ADC device

Raspberry pi 3 is used as a middle level controller which captures auxilary camera feed. It controls the eyes which is an LED strip arranged as a 2D matrix. The LED strip is modelled as a 2D matrix and the elements of the matrix are modified to output the desired colors in the eyes. The pi also performs the closed cloop control parameters to move the arms and the head according to the change in the environment. It also relays the information from the jetson to the low level controller.
